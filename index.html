Our Goals.

Here's what we want LlamaOS to be

Fully Open Source

Being Open Source is very important to us, because it means that anyone can view the code and contribute to it, if the person has any ideas. We want to make it as easy as possible for other developers to contribute to LlamaOS. Just fork the project on Github and open a Pull Request and we will take a look at it.

Linux Kernal

LlamaOS will be based on the Linux Kernal, so Linux Programs will run on LlamaOS too!

Clean Code

We will keep our Code very good commented and clean, so that other devs can easily understand what is going on and why.

New features

We don't just want to create a normal OS. LlamaOS will have a lot of new features that do not currently exist in other Operating Systems. We will only ADD features, not take them. Don't worry, we will always keep the earphone plugs on your pc working just as they always worked ;)

Interact with users

We will try to respond to every E-Mail we get and read everything on Github, so feel free to contact us!
